## Description
Queen-led collective intelligence with Byzantine, majority, and weighted consensus mechanisms, eight worker specializations, and persistent collective SQLite memory with LRU caching.

## Overview
Distributed Consensus Coordination (formerly Hive Mind Coordination) is a queen-led collective intelligence system for multi-agent orchestration. A central "queen" agent coordinates specialized worker agents through robust consensus mechanisms, ensuring reliable task execution and decision-making in complex environments.

## Key Capabilities
- **Multi-Mechanism Consensus**: Support for Byzantine fault-tolerant, majority-based, and weighted voting protocols.
- **Role-Based Specialization**: Coordination across multiple distinct agent specializations (e.g., researchers, executors, critics).
- **Persistent Collective Memory**: Shared state management with high-performance caching for associative retrieval.
- **Fault Tolerance**: Resilience against individual worker failures or inconsistent agent outputs.
